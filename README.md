# Clothify
Clothify Mobile App.

# Features
* Buy and sell clothes online
* Message other store
* Rate others product
* Search other store or product

# Course Overview
* Basic React Native Components
* Database Design using Appwrite
* User Registration & Authenticaiton
* Deployment

# Tech Stack
* React Native
* AppWrite
* HTML, CSS and JS
